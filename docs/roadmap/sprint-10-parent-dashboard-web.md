# Sprint 10 - Parent Dashboard Web

## Goal

Build the first React Web parent dashboard using reusable components.

## Scope

1. React Web project foundation.
2. ParentDashboardPage.
3. StudentSelector.
4. PerformanceSummaryCards.
5. ActivityHistoryChart.
6. SubjectPerformancePanel.
7. LearningGapList.
8. CurrentActionPlanCard.
9. RecentActivityTimeline.
10. API client integration.

## Acceptance Criteria

- Parent dashboard renders from API response.
- Dashboard supports empty states.
- Dashboard shows weekly summary.
- Dashboard shows activity chart.
- Dashboard shows subject performance.
- Dashboard shows gaps.
- Dashboard shows current action plan.
- Components are reusable and typed.
- No mobile app work is included.

## Out of Scope

- React Native.
- Expo.
- Full admin UI.
- Complex design system.
- Gamification visuals.
