# Release Plan

## Release 0.1 - Technical Foundation

Goal:

Prove backend and local infrastructure.

Includes:

- Docker local infrastructure.
- PostgreSQL.
- Redis.
- Nginx.
- Spring Boot backend.
- Actuator health.
- Basic system API.

Status:

Completed / in progress.

## Release 0.2 - Education Foundation

Goal:

Create the first real education domain model.

Includes:

- Student.
- Subject.
- Curriculum framework.
- Learning objective.
- Basic CRUD APIs.
- Flyway migrations.

## Release 0.3 - Configurable Learning

Goal:

Allow flexible learning profiles and admin-controlled freedom.

Includes:

- Learning policy.
- Teaching method.
- Student learning profile.
- Subject enrollment.
- Learning goals.
- Policy-based validation.

## Release 0.4 - API-First Onboarding

Goal:

Allow admins to onboard users through templates and APIs.

Includes:

- Import template.
- Import batch.
- CSV upload.
- Validation errors.
- Preview.
- Confirmation.
- Pending users.
- Signup invitations.

## Release 0.5 - Signup Activation

Goal:

Allow pre-registered users to activate accounts safely.

Includes:

- Identity document.
- Contact channel.
- Email verification.
- Phone verification.
- PIN creation.
- Account activation.

## Release 0.6 - Gamification Foundation

Goal:

Create a healthy engagement loop for students.

Includes:

- Gamification profile.
- Points ledger.
- Daily streak.
- Basic achievements.
- Avatar profile.
- Policy to enable/disable gamification.

## Release 0.7 - Learning Analytics

Goal:

Provide meaningful progress visibility.

Includes:

- Daily summary.
- Accuracy trend.
- Time-on-task.
- Gap signals.
- Strength signals.
- Guardian dashboard API.

## Release 0.8 - AI Assistance

Goal:

Introduce AI behind an abstraction layer.

Includes:

- AI gateway.
- Prompt templates.
- AI audit log.
- Activity generation draft.
- Explanation generation.
- Remediation recommendation.
