# Product Story Map

## Backbone

```text
Configure platform
  -> Onboard users
    -> Activate accounts
      -> Configure learning
        -> Run learning activity
          -> Track progress
            -> Motivate student
              -> Personalize with AI
```

## Activity 1 - Configure Platform

Stories:

- Define tenant/organization.
- Configure learning policies.
- Configure available subjects.
- Configure available teaching methods.
- Enable/disable gamification.
- Enable/disable student autonomy.

## Activity 2 - Onboard Users

Stories:

- Download user import template.
- Upload user import file.
- Validate imported records.
- Preview valid records.
- Confirm import.
- View import history.
- Create signup invitations.

## Activity 3 - Activate Accounts

Stories:

- Start signup using primary document.
- Verify email.
- Verify phone.
- Create PIN.
- Activate account.
- Expire stale invitations.
- Reissue invitation.

## Activity 4 - Configure Learning

Stories:

- Create student learning profile.
- Select teaching method.
- Select subjects.
- Create learning goals.
- Apply policy checks.
- Guardian/admin override.

## Activity 5 - Run Learning Activity

Stories:

- Create daily activity.
- Present questions.
- Submit answers.
- Capture time.
- Capture confidence.
- Score attempt.

## Activity 6 - Track Progress

Stories:

- Generate daily summary.
- Track accuracy.
- Track time-on-task.
- Detect gaps.
- Detect strengths.
- Track mastery.

## Activity 7 - Motivate Student

Stories:

- Award points.
- Track streaks.
- Unlock achievements.
- Customize avatar.
- Show personal progress.

## Activity 8 - Personalize with AI

Stories:

- Generate activity draft.
- Generate explanation.
- Generate remediation.
- Recommend next activity.
- Summarize progress for guardian.
